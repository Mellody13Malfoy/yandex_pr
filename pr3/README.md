у основного алгоритма faust в файле app.py есть две дополнительные функции, две функции агента и 5 функций для димонстрации
дополнительные функции
red_text - нужен зашифровки слов в сообщении
is_blocked - проверка на блокировку пользователя
функции агентов
process_coomands - запускается перед осноной обработки для обработки команд на добавления слов и пользователей в стоп листы
process_message - основная функция обработки сообщения
остальные функции нужны для димонстрации состояния, жизни системы и димонстрации заблокированных пользователей.
у producer.py есть только функция main в которой происходят отпрапвка сообщений в топик
у consuner.py есть функция проверки подключения кафки, функция проверки существования топика, и чтения посленего топика с пропущенными сообщениями
