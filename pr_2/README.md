есть 3 файла Python внутри класа KafkaProduserApp(1 продюсер и 2 консьюмера)
продьюсер имеет в себе функции:
deliver_report - состояние доставки
produce_messages - создает сообщение и отправляет его с задержками 
1 консьюмер single
process_single_message - обработка сообщения
consumer - функция старта консьюмера и потребления по одному сообщению 
2 консьюмер batch
process_batch - обработка сообщений
consume - функция старта консьюмера и потребления 10 сообщений сразу
